## üîÑ How to Switch Themes

If you want to try a different option:

1. Open `qadeem-eseek-frontend/src/theme/theme-colors.js`
2. Copy the color values from your chosen option above
3. Replace the current values in the `primary` and `secondary` exports
4. Save and the changes will reflect immediately
